# wolf3d @ 42

A Wolfeinstein-like renderer made in C using minilibX, with texture and basic
lighting support.

## Compiling
Run `make` and you're set. Compiles on Mac OS X and Linux (with XServer).

## Running
Run `./wolf3d [map_file]`, a window should appear.
